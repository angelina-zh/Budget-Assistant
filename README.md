# Budget-Assistant
Программа хранит ежедневные расходы пользователя за месяц. Пользователь может добавить расход (день, сумма, категория) или запросить сумму расходов за период с дня A по день B. Для ускорения ответов на такие запросы построен массив префиксных сумм (накопительная сумма расходов по дням). Ответ на запрос — за O(1).
